# Product Overview

**Trading Agent** is a weekend class project exploring agentic/vibe coding techniques applied to automated trading.

The project is an AI-powered trading agent that can analyze market data, make trading decisions, and potentially execute trades. It is intended as a learning exercise in building LLM-driven autonomous agents within a financial domain.

## Core Goals

- Build an autonomous agent capable of reasoning about market conditions
- Demonstrate agentic coding patterns (tool use, planning, memory, reflection)
- Keep the codebase simple, readable, and iterative — this is a learning project

## Audience

- The primary user is the developer/student building and experimenting with the agent
- Not intended for production trading use
